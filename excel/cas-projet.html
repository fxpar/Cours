Convertissez les numéros de projet

Utilisez une formule pour transformer les numéros des projet en véritable "code projet".

Regardez dans la feuille "projet", vous devriez obtenir quelque choses de similaire.

Pour convertir un numéro en texte formaté vous pouvez utiliser la merveilleuse fonction TEXT(number,"format") qui fera tout le travail. Elle adaptera le nombre de zéro à la taille du nombre.

Comme ici le code est simple, vous pouvez utiliser: =TEXT(number,"P00000") pour créer le code du projet.

Si le code avait été plus complexe, une concaténation en dehors de Texte aurait été préférable: ="P"&TEXT(number,"00000")
