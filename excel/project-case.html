Convert Project Number

Convert the number of the project into a "P" followed by 5 digit.

It will be very useful when you will want to get informations from the project sheet.

To turn a number into a formatted text you ca use the fantastic TEXT(number,"format") function. It would adapt the number of zeros to the number of characters in the project number.

Because the code is simple, you could do: =TEXT(number,"P00000") to create the project code.

But if the code was more complex, a concatenation outside the text formatting could also be possible: ="P"&TEXT(number,"00000")
